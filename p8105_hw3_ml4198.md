## Problem 1

``` r
library(p8105.datasets)
data("instacart")
```

This is a dataframe of Instacart orders with 17 variables and 1,384,617
entries. Key variables are user\_id, order\_id, product\_id, aisle\_id.

``` r
instacart %>% 
    group_by(aisle_id) %>% 
  summarize(
    n_obs = n()
    ) %>% 
  mutate(
    freq_rank = min_rank(desc(n_obs))
    ) %>% 
  filter(freq_rank < 10)
```

``` r
instacart %>% 
    group_by(aisle_id) %>% 
  summarize(
    n_obs = n()
    ) %>% 
  mutate(
    freq_rank = min_rank(desc(n_obs))
    ) %>% 
  filter(n_obs > 10000) %>% 
  arrange(freq_rank) %>% 
  ggplot(aes(x = freq_rank, y = n_obs)) +
  geom_point() + 
  scale_x_continuous(
    breaks = c(1, 5, 10, 15, 20, 25, 30, 35),
    minor_breaks = c(1:39)) +
  labs(
    x = "Aisle",
    y = "Number of items ordered"
  )
```
